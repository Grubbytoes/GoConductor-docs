MusicSwitch

Class

A controller node designed such that, at any given time, only one of it's children should be playing at any one time. Intended for use switching between tracks.

Extends: MultiTrackPlayer

Properties

  • GcNode CurrentlyPlaying

    Whichever of the switch's children is currently playing. Ie, whichever one was last cued. By default, this is null.

  • Boolean RestartOnRecue

    Determines the switches behavior if 'cue' is called, passed with the same track that is already playing. If false, the call will simply ignored; if true, the track will be started from the beginning, as though it is a new (although identical) track.

  • Float TransitionTime

    The time, in seconds, the crossfade between two tracks will take when 'cue' is called during playback. If set to 0, this gives the impression of a hard cut.

Methods

  • Boolean Cue(GcNode track)

    Assigns track to CurrentlyPlaying. If another track is playing when this method is called, a transition between the old track and the new is triggered.
