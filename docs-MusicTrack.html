MusicTrack

Class

One of the simplest track controllers added by GoConductor, still geared towards playing music. It is designed to be a leaf node of a Switch/COnductor tree, but can also be used on it's own. Essentially adds as a wrapper for a Godot AudioStreamPlayer with some streamlined functionality.

Extends: GcNode

Properties

  • Boolean AutoPlay

    If true, the music track will automatically play when the scene it is part of is ready. This overrides any control from nodes higher up the scene tree (Conductors, Switches etc.)

  • Boolean Loop

    If true, the track will automatically restart upon reaching the end. This will appear as a seamless loop (assuming the audio itself is clipped accordingly). This will only work if the AudioStream is imported with loop set to false.

  • AudioStream Track

    The audio resource this code is meant to play. Note that if you desire the track to loop, it should be imported with loop set to false, this is because GoConductor handles looping internal (see Loop property above).

  • Float Gain

    Applies a boost or reduction to the track's volume when it is played. This property is also used for fading tracks in/out using Tweens, so is designed to be changeable at runtime.
