MultiTrackPlayer

Abstract Class

This is the base class for any track controller node which acts as director for one or more child nodes, two examples being the MusicSwitch and the MusicConductor. These nodes will have no functionality unless they have children to control.

Extends: CgNode

Extended By: MusicConductor, MusicSwitch

Methods

  • Boolean Cue(GcNode track)

    Cues a track in (or out). The explicit aliases for this function with different parameters as opposed to just simple overloads, given below, exist because GDScript does not support overloading for cross-language-scripting. Thus, it is simpler to call explicitly different methods from .GD scripts, rather than having to call get_child(idx) or find_child(name) whenever needed.
    Returns: true if the track is cued successfully.

  • Boolean CueIdx(int trackIdx)

    Calls Cue for the track found as a child node at the given index.
    Returns: true if the track is found and cued successfully.

  • Boolean CueName(String trackName)

    Calls Cue for the track found as a child node with the given name. Note that it does not search recursively (ie for grandchildren etc.)
    Returns: true if the track is found and cued successfully.
