GoConductor

Singleton

The first thing you should do (and only thing you really need to do) upon adding GoConductor to your project structure is add GoConductor.tscn as an autoload scene in your project settings. By default it should have the name GoConductor, but everything will still work if you want to load it under a different name.

Extends: Godot Node

Methods

  • Boolean load_stage(string stage_path, string pseudonym = none)

    Loads the stage of the given path relative to the GoConductor\Stages directory. Path does NOT need to include '.tscn'. Will load the stage under a pseudonym, if one is given, which it can be accessed as in future. Returns: True if the stage is loaded successfully

  • Boolean unstage(string stage_name)

    Unloads the stage of the given name, if it has been loaded. Returns: True if the stage is found and removed successfully

  • Boolean swap_stage(string new_stage_path, string stage_name)

    unloads the stage given by name, and loads the new stage as given by the path under the pseudonym of the original stage. If the original stage cannot be found, this function does nothing. Returns: True if the new stage is loaded and swapped in successfully

  • Variant get_stage_pointer(string stage_name)

    Gets whatever is in the given stage. if the stage has only one child node, it will be this. If it has more than one, it will return an ordered array, OR a name dictionary, depending on the stage's settings. Returns: The contents of the stage under the given name, or null if one does not exist/has not been loaded

  • BaseStage get_stage(string stage_name)

    Passes a reference to the stage root node. Note that the base stage itself does not have any useful public methods; this is intended for users extending the stage script to add their own functionality. Returns: The instance of the stage itself under the given name, or null if one does not exist/has not been loaded

  • Void mute_music(boolean mute=true)

    Mutes (if mute=true, otherwise will unmute) the audio bus marked for music in GoConductor\settings.txt

  • Void mute_sfx()

    Mutes (if mute=true, otherwise will unmute) the audio bus marked for sound effects in GoConductor\settings.txt
