TrackController

Interface

This is the basic interface which covers nearly every new class added by GoConductor. In short, it is anything which has control over audio, and as such requires simple commands like Play, Pause, Stop, etc. Picture the buttons on a CD player, or any other media player, and you'll get the idea.

Implemented By: GcNode
