GcNode

Abstract class

The most basic implementation of the TrackController interface, and the type from which most others inherit. Is not meant to be instanced. Exists as a common base type allowing different subtypes and external classes to access and communicate with each other. Most often used for parameter and return typing.

Extends: TrackController

Extended By: MultiTrackPlayer, MusicTrack

Properties

  • Float PlaybackPosition

    The position in playback the track is currently at (or will be when it is next played). In seconds, where 0.0 seconds is the beginning of the track.

  • Boolean Playing

    Whether or not the track is currently playing.

Methods

  • void Play()

    Plays the audio from where it was last paused, or from the beginning if stopped or not played/playing. Sets Playing to true.

  • void Pause()


    Pauses the audio, ready to be played again from that same position. Sets Playing to false.

  • void Stop()


    Stops the audio, to be played again from the false.

  • void Restart()


    Plays the audio from the beginning of the track. Sets Playing to true.

  • void TogglePause()


    Plays the audio if it is not currently playing, or pauses it if it is. Sets Playing accordingly.

  • void PlayFrom(float position)

    Plays the audio from the position given. Sets Playing to true.
